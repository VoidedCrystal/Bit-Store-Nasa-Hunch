The Problem:

Students participating in NASA HUNCH have been using USB Drives to store their projects. There are many risks when it comes to handling USB drives. For example, USB drives can be lost, stolen, or damaged. This means all of the students work can be lost.

Our Solution:

We will create a data repository that will store NASA HUNCH projects safely and effiecently. Our repository will allow the upload and download of files up to 10 GB. We'll create a safe authentication method to ensure nobody can get unauthorized access to your account in the case of a databreach. To prevent databreaches we will create a firewall to prevent hackers from entering our network.

Risks:

There are many risks when it comes to a project like this that can be very deterimental. Things like hacking, file corruption, and file size can pose problems for the project data. We plan on creating our own firewall in order to keep hackers away from our network. We will create an automated back-up system to keep files safe if anything goes wrong with them. We'll also implement a 10 GB limit on file size to ensure files can be uploaded and downloaded smoothly
